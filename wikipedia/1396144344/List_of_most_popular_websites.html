Ranking measures

Alexa traffic rank

Alexa Internet ranks websites based on a combined measure of page views and unique site users. Alexa creates a list of "top websites" based on this data time-averaged over three-month periods.[1]

Linking root domains

The number of linking root domains is a measure of how many external sites link to the website in question.

Google Display Network Ad Planner

The Google Display Network Ad Planner measures the number of unique visitors, for use by Google's advertisers.
