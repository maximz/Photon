This is a list of Xbox One games currently planned for release either at retail or via download.

There are currently 143 games on this list.

Exclusive = 17
Microsoft exclusive = 7
Console exclusive or timed = 7/3
Multiplatform = 105
Playable = 35 available to purchase (9 of these are exclusive to Xbox One)

Xbox One games[edit]

Exclusivity[edit]

  • Yes = Exclusive only to this computing platform.
  • Microsoft = Exclusive to Microsoft's platforms.
  • Console = Not available on competing video game consoles, but is available on PCs or other platforms.
  • Timed = Confirmed as exclusive for a certain period of time, but will become available on other platforms later.
  • No = Available to more than one console of this console generation.

Kinect Support[edit]

  • Yes = Exclusive control via the Kinect.
  • Supported = Control is primarily with the controller, but the Kinect is usable for the game.
  • No = Control is with the controller exclusively.

Demo[edit]

  • Yes = Downloadable demo available.
  • No = No downloadable demo available.
This is an incomplete list, which may never be able to satisfy particular standards for completeness. You can help by expanding it with reliably sourced entries.
